# アーキテクチャ選定について


- パターンは、Platform・作るアプリによって変わってくることがある
  - 全てに通用する完璧なパターンはない
- 根源となるClean Architectureは入れる

- 両方適用できるなかで、reduxパターンかMVVMパターンが有力候補
- TCAはiOS発祥なのだが、Androidにも適用できる
  - [GitHub - toggl/komposable-architecture: 🏗️ Kotlin implementation of Point-Free's composable architecture](https://github.com/toggl/komposable-architecture)
  - ただし、Android側での利用実績は少ない
