# Xcode with Claude


## Setup

### Claude Code
https://jobirun.com/claude-ai-in-xcode/
有料プランに申し込む

### Xcode
Apple Intellgenceがないと、XcodeでClaudeを使えない
https://www.apple.com/jp/apple-intelligence/
- macOS Tahoe 26.0以降
- Xcode 26以降

Xcode > Settings > IntellgenceからClaudeにログイン


## 試す

Xcodeの左上あたりのアイコンで「Coding Assistance」があるので、そこでプロンプトを入力。
プロンプトのやり方の問題かと思うけど、バイブコーディングは向かないかも？
今、できそうなことを考えてみる
- 不明なエラーの解消
- アーキテクチャの解析
- バグになりそうなところの解析
- テスト書く

逆に、実装中はチャット機能よりも「予測」の機能が使えそう。
