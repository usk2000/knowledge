# .navigationBarTitle deprecated


https://developer.apple.com/forums/thread/650694

`.navigationBarTitle`はdeprecatedになった。
代わりに`.navigationTitle`を使用する。
https://developer.apple.com/documentation/swiftui/view/navigationtitle(_:)-43srq


```swift
struct ContentView: View {
    var body: some View {
        NavigationView {
            List {
                NavigationLink("Scan", destination: ScanView())
            }
            .listStyle(GroupedListStyle())
            .navigationTitle("CoreBluetooth")
            .navigationViewStyle(StackNavigationViewStyle())
            
        }

    }
}
```

→エラーが出る
```
[LayoutConstraints] Unable to simultaneously satisfy constraints.
...
```

struct ContentView: View {
