# ライフサイクル


[iOSアプリのライフサイクル - Qiita](https://qiita.com/KenNagami/items/766d5f95940c76a8c3cd)

アプリが未起動時にプッシュ通知を押した場合
- func application(_:willFinishLaunchingWithOptions)
- func application(_:didFinishLaunchingWithOptions:)
- func userNotificationCenter(_:didReceive:withCompletionHandler:)
- func applicationDidBecomeActive(UIApplication)

アプリがバックグラウンド状態でプッシュ通知を押した場合

- func applicationWillEnterForeground(UIApplication)
- func userNotificationCenter(_:didReceive:withCompletionHandler:)
- func applicationDidBecomeActive(UIApplication)

どちらの場合も、userNotificationCenter:didReceiveの時はapplicationStateはinactive
