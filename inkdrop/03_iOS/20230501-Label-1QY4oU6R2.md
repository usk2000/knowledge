# Label



[Change the Icon Color in the Label View | Medium](https://medium.com/@randomdinodev/changing-the-image-icon-color-in-a-label-d6ad1df4202f)

Labelのアイコンの色をtintにするには

```swift
                    Label {
                        Text("記録").font(.myFont(size: 18))
                    } icon: {
                        Image("IconRecord")
                            .renderingMode(.template)
                    }.labelStyle(TabLabelStyle())   
```
