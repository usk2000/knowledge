## インストール
https://www.rust-lang.org/ja/tools/install

```
% curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
....
1) Proceed with installation (default)
2) Customize installation
3) Cancel installation
# 1を選択
```
その後新しいシェルにする。

アップデート
```
% rustup update
```

% rustc --version
