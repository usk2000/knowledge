## 方法２
https://qiita.com/tutuz/items/af4ab620f47b8a56ed55
プロジェクトごとに設定する方法

gradle.properties
```
org.gradle.java.home=/Library/Java/JavaVirtualMachines/jdk-11.0.5.jdk/Contents/Home
```


これでリスタート
./gradlew --stop
→できないので、gradleをダウングレード

これ設定しているからかな？
```
compileOptions {
        sourceCompatibility JavaVersion.VERSION_1_8
        targetCompatibility JavaVersion.VERSION_1_8
    }
```
→これをコメントアウト
うまくいかなかった。

やはりgradle7.0以上で全体を上げるしかないかな。

```
compileOptions {
        sourceCompatibility JavaVersion.VERSION_11
        targetCompatibility JavaVersion.VERSION_11
    }
    kotlinOptions {
        jvmTarget = '11'
    }
```
