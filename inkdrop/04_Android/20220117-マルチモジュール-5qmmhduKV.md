モジュール間でのバージョン管理
[Sharing dependency versions between projects](https://docs.gradle.org/current/userguide/platforms.html)

versionCatalogsを使う場合は、以下の方法が必要。
https://docs.gradle.org/7.0.2/userguide/feature_lifecycle.html#feature_preview

```groovy
enableFeaturePreview("VERSION_CATALOGS")
dependencyResolutionManagement {
    repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
    repositories {
        google()
        mavenCentral()
        jcenter() // Warning: this repository is going to shut down soon
    }
    versionCatalogs {
        libs {
            alias("core-ktx").to("androidx.core:core-ktx:1.7.0")
        }
    }
}
```

これ結構便利だな。
