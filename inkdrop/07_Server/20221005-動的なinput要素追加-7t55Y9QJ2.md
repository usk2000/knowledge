# 動的なinput要素追加



動的にinput追加した際、form.submitを押さないと追加したinputが反映されない

```pug

form(method="POST", action="/reports")
  input(type="hidden", name="type", value="hogehoge")
	//- input(type="hidden", name="id", value="brbr")

```
`id`のところを動的に追加しても、通常だと値がBackendに反映されない

```pug

form(method="POST", action="/reports", onsubmit="reportAction(event)")
  input(type="hidden", name="type", value="hogehoge")
	//- input(type="hidden", name="id", value="brbr")
  
  
script.
	function reportAction(event) {
		event.preventDefault()

		let form = document.querySelector('form')
		form.submit()

	}
```
