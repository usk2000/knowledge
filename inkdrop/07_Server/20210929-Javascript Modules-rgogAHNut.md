# Javascript Modules

[JavaScript modules](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Modules)

## 方法

Javascript側
- import形式を使う
- exportで変数、関数をエクスポートする
- HTML側のscriptタグにtype=moduleを指定する
- jQueryなどでアクションにバインドする。
