# DB構造（第１フェーズ）

## ユーザ（/@:id）
:idは数字、スクリーンIDのどちらでも
- ID(id)
- ニックネーム(name)
- スクリーンID(screen_id)
- 肩書き(title)
- 画像（複数・最大５枚）(images)
- 自己紹介(introduction)
- 作成日時(created)
- 更新日時(updated)
- ペット情報(pets)
  - ニックネーム(name)
  - スクリーンID(screen_id)
  - 画像（複数・最大５枚）（images）
  - 自己紹介（introduction）
  - プロフィール項目(profiles)
    - 項目(type)
    - 名称(text)
    - ユーザー追加かどうか(user_added)
    - 表示・非表示(shown)
    - リンク(link)      

## ペット(/@:owner?p=[:id])→なし

## 保存(stocks)
- 保存者ID（user）
- リソースタイプ (itype): items, users
- 保存対象ID (id)
- 日時（date）
- 名称 (name)
- 画像 (image)
  - パス (path)
  - URL (url)

## 商品(items)
- ID(id)
- 名称(name)
- 画像（複数）(images)
- 評価（運営）(review_by_manager)
- レビュー数（初期は0）(review_count)
- 商品紹介（Markdown可能）(text)
- 公式ホームページ（homepage）
- links
  - type: amazonとか
  - link: ページ
- infos
  - type: 価格とか
  - text: 名称とか
  - link: リンク（あれば）
- created: Date
- updated: Date

## フォロー(followers)
- ID(id)
- フォローする人(from)
- フォローされる人(to)
- フォローされる人の名前(to_name)
- フォローされる人の画像(to_image)

## 設定(settings)
- ID(id)（ユーザーID）
- メール通知するかどうか（mail）

## 招待(invites)
- ID(id)
- 招待する人（from）
- 招待できる最大人数 (max)
- 招待コード
- 招待された人たち（inviteds）


## お知らせ
- ID
- タイトル(title)
- 説明文(text)→Markdown可能
- created
- updated
