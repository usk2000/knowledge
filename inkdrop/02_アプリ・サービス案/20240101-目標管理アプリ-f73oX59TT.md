# 目標管理アプリ

- 目標の達成率を視覚化する


- 目標と数値を管理
  - カウントアップ、カウントダウン
  - 数値の推移
  - 数値化できないもの、例えば曲を覚えるとかはどうするか
    - タスク細分化にする
- 日数ベース、月に何回とか
- イベントベース、TOEICとか
- バーンダウンチャートがあると面白い
  - 期限が必要
- 日課はどうするか、継続日数つまりカウントアップか。継続週数もあるかも
- タグ・サブタスクもサポート？
  - タグにする。ソートして表示はできるようにする。
- メモも入力できるようにする
- まだ時期が決まっていないものも入力できるように
- 時間の入力と回数の入力、その日付も
- 進捗がわかる表示が良い
- 単純タスクも良いかも

- 英会話
  - レッスン復習・新しいレッスンを受ける
  - 復習：開始時間と必要時間
  - 新しいレッスン：開始時間と終了時間と必要回数
- TOEIC
  - L&RとS&W
  - 開始時間と必要時間
- visionOS
  - 開始時間と必要時間


- 時間積み上げタイプ：開始時期と必要時間、あとは終了時期
- 期限と回数タイプ：開始・終了時期、必要回数
- 単純タスク：開始・終了時期のみ
- リストとカレンダー画面
  - リスト：目標・ゴールの確認


回数タイプの中には、時間の方がしっくりくることがある。切り替えの実装はせず、削除してから追加にする。
時間でおおよその見積もりを入れる。

レコードを追加したら、並べ替えても良いかも。
