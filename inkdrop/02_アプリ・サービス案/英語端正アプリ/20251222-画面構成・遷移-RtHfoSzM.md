# 画面構成・遷移


- **Onboarding**
  - OB1: Welcome
  - OB2: How it works
  - OB3: Microphone Permission
- **Home**
  - HOME: Recording
  - RESULT: Result Summary
  - SCORE: Fit Score Detail
  - ADVICE: Fit Advice Detail
- **Practice**
  - PRACTICE\_TOP
  - DRILLS\_LIST
  - DRILL\_SESSION
  - TALK (Fit Talk) ※MVP後半 or 次版でもOK
- **Log**
  - LOG\_LIST
  - LOG\_DETAIL
- **Settings**
  - SETTINGS\_TOP
  - PROFILE
  - PRO (Paywall)
  - HELP


---

### 初回起動フロー

- `App Launch`
  - 初回 → `OB1 → OB2 → OB3(マイク許可) → HOME`
  - 2回目以降 → `HOME`

### HOME（録音中心）

- `HOME`
  - 録音開始 → `RECORDING (同一画面内状態)`
  - 録音停止 → `RESULT`
- `RESULT`
  - `See Score` → `SCORE`
  - `See Advice` → `ADVICE`
  - `Try Again` → `HOME(録音状態へ)`
  - `Save`（自動保存でも可）→ `LOG_LIST`（任意導線）
- `SCORE`
  - 戻る → `RESULT`
- `ADVICE`
  - `Retry` → `HOME(録音状態へ)`
  - 戻る → `RESULT`

### PRACTICE（練習導線）

- `PRACTICE_TOP`
  - `Drills` → `DRILLS_LIST → DRILL_SESSION → RESULT`
  - `Fit Talk` → `TALK → RESULT`（MVP後追加でもOK）
- セッション結果は **同じ RESULT へ統一**  
  → スコア表示とアドバイス表示の体験が一貫します。

### LOG（履歴）

- `LOG_LIST`
  - item tap → `LOG_DETAIL`
- `LOG_DETAIL`
  - `Replay` →（再生）
  - `Retry this` → `HOME(録音状態へ)` または `DRILL_SESSION`

### SETTINGS（課金・ヘルプ）

- `SETTINGS_TOP`
  - `Profile` → `PROFILE`
  - `Pro` → `PRO`
  - `Help` → `HELP`
