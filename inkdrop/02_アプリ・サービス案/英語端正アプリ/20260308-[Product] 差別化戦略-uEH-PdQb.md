# [Product] 差別化戦略

Accent Fit を **実際にユーザーが選ぶプロダクトにするための差別化戦略**を整理します。  
発音アプリ市場はすでに競合が多いので、**単なる発音評価アプリでは勝てません**。重要なのは「何を主価値として打ち出すか」です。

***

# 1. 市場の代表的な競合

主要な発音・英語学習アプリを整理します。

| アプリ        | 特徴           |
| ---------- | ------------ |
| ELSA Speak | AI発音評価の代表    |
| Speak      | AI会話練習       |
| Duolingo   | gamification |
| Cake       | 動画学習         |

これらを見ると、主な方向は3つです。

```
AI発音分析
会話練習
ゲーム学習
```

***

# 2. Accent Fit の立ち位置

Accent Fit は **次のポジションが最も強いです**。

```
「短時間の発音トレーニング特化アプリ」
```

つまり

```
10秒発音トレーニング
```

です。

***

# 3. 差別化コンセプト

Accent Fit のコアメッセージ

```
Speak Clearly, Naturally
```

具体化すると

```
1日1分で発音改善
```

です。

***

# 4. 差別化ポイント

Accent Fit の差別化は **4つの柱**です。

| 差別化                  | 内容    |
| -------------------- | ----- |
| Ultra short practice | 10秒練習 |
| Pronunciation focus  | 発音特化  |
| Immediate feedback   | 即スコア  |
| Weakness training    | 弱点補強  |
