# ChatGPT用の指示


## チャットでの説明

上記内容を以下形式でまとめてください：
- Decision
- Open Questions
- Insights
- Next Action

## 各項目の説明
- Decisions
  - 「もう決めたこと」
  - 必ず理由を含める
- Open Questions
  - 未解決の論点
  - 1行で書く
- Insights
  - 気づき・設計原則
  - 再利用価値があるもの
- Next Actions
  - 次にやる具体タスク
