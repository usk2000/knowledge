# [Product] AccentFitのUX

Accent Fit は **「練習→評価→改善」を何度も回すアプリ**なので、UX設計の目的は次です。

```
ユーザーが毎日1〜3分でも練習を続けられること
```

そのため **摩擦を徹底的に減らすUX**にします。

***

# 1. Accent Fit UXの基本原則

UXは次の5原則で設計します。

| 原則                | 内容        |
| ----------------- | --------- |
| Instant start     | すぐ録音できる   |
| Short loop        | 30秒で1回完結  |
| Clear feedback    | スコアがすぐ出る  |
| Small improvement | 小さな改善を感じる |
| Daily habit       | 毎日続ける     |

***

# 3. 起動UX


# 4. Recording UX

録音は**最も重要な画面**です。

UI

```
Could you send the report?

🎙 Recording...

[ Stop ]
```

UXルール

- ワンタップ録音

- 自動停止OK

- 再録音可能

***

# 5. Result UX

結果は**一瞬で理解できる**必要があります。

例

```
Fit Score
82

Rhythm   80
Stress   75
Clarity  90
```

重要なのは

```
1つだけAdvice
```

例

```
Try stressing the word "report".
```

***
