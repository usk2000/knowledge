# Idea

アプリ・サービスアイディアを、あとから検討・検証しやすい形で残すための目次です。

## 記事

- [音声から楽譜を生成する MCP サービス](audio-to-score-mcp-service.html)

## 配置方針

- アプリ、Web サービス、プロダクト、ツール、事業仮説などのアイディアを置きます。
- 1ページ1アイディアを基本にし、思いつき、背景、想定ユーザー、価値仮説、MVP、検証方法、未決事項を分けて整理します。
- 技術調査が主題になった場合は `ai/`、`apple/`、`android/` などの調査カテゴリに分けることも検討します。
